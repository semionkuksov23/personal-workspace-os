# Command: Implement

## Purpose
Execute a previously confirmed plan.

## Preconditions
- Confirmed plan exists and is final.

## Steps
1. Restate confirmed plan briefly.
2. Execute tasks sequentially.
3. After each major step:
   - verify output
   - update current context files if project state changed
4. If blockers appear:
   - resolve if possible
   - otherwise ask focused question
5. Run final validation:
   - file placement and naming
   - ID consistency
   - chronology/strategy/analysis cross-references
6. Deliver completion summary with residual risks.

## Output
- What changed
- What was validated
- Remaining follow-ups
