# FileAnalysisArchive

## Purpose
This file stores document analysis entries for completed projects moved from Current via the `Archive` command.

## Rules
- Preserve IDs and original analysis text.
- Preserve all project tags and cross references.
- Keep paths in backticks and dates in ISO format.

| Archive Action Date | `<YYYY-MM-DD>` |

| Type | `<Type>` |
| Status | `<Final status at archive time>` |
